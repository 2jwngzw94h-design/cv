# Site professionnel — Antoine Courtin

Site statique prêt pour GitHub Pages.

## Publication sur GitHub Pages

1. Pousser ce dépôt sur GitHub.
2. Ouvrir les réglages du dépôt.
3. Dans « Pages », choisir la branche courante et le dossier racine.
4. Valider : `index.html` devient la page d’accueil.

## CV en PDF

La page `cv.html` est préparée pour l’impression. Ouvrir la page dans un navigateur, lancer l’impression, puis choisir « Enregistrer en PDF ».
